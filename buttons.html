Brand Button

Google and Facebook buttons are available featuring each company's respective brand color. they are used on the user login and registration pages.

You can create more custom buttons by adding a new color variable in the _variable.scss file and then using Bootstrap button variant mixin to create a new style, as demonstrated in the _buttons.scss file.

.btn-google .btn-facebook
